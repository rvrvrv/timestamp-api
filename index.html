Timestamp Microservice API Project for FreeCodeCamp

User stories:

  • I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 16, 2017).
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example query usage:

https://timestamp-api-rv.herokuapp.com/January%2016,%202017
https://timestamp-api-rv.herokuapp.com/1484524800

Example query output:

{
  "unix": 1484524800,
  "natural": "January 16, 2017"
 }
